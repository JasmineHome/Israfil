# IsrafilCLI

## IsrafilCLI 编译
### 准备环境 
Windows: 
 - 从golang官网(golang.org)(GFW Certified,需要科学上网)下载安装

Linux:
 - pacman -Syu
 - pacman -S golang

OS X: 
 - brew install go

### 从源码编译
 - go get github.com/LER0ever/Israfil/HttpAPI
 - cd $GOPATH/src/github.com/LER0ever/Israfil/HttpAPI
 - ./build.sh # build.bat for Windows
